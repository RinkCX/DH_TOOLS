## 📖 简介

这是一个专为 GM 设计的“一页流”战役管理仪表盘。它将战役信息、跑团日志、遭遇战追踪和卡片数据库整合在一个简洁的网页中。

**核心理念：**
*   **💾 纯本地运行**：利用 IndexedDB 技术，所有数据存储在你的浏览器中，无需担心服务器关闭或数据泄露。
*   **⚡ 极速响应**：单文件 HTML，秒开，无外部字体依赖，国内网络环境友好。
*   **🛠️ 高度自由**：支持自制（Homebrew）内容，可自定义卡片所属源、倒计时和计数器。

## ✨ 核心功能

### 🏠 概览 (Overview)
- **战役仪表盘**：记录模组名、基调、玩家信息等核心设定。
- **一键招募**：填写信息后，可一键复制标准格式的“挂团/招募”文字公告。
- **自定义横幅**：支持上传战役封面图，并可自由拖拽位置和缩放。

### 📜 章节 (Sessions)
- **日志记录**：记录每次跑团的日期、参与玩家 (PL) 和剧情纪要。
- **智能链接**：自动识别文本中的 URL 并转化为可点击链接。

### 👹 遭遇 (Encounters) - 战斗核心
- **环境管理**：支持选择场景卡，置顶显示场景趋向和特性。
- **敌人追踪**：可视化追踪敌人的 **生命点 (HP)** 和 **压力点 (Stress)**。
- **战术速查 (Quick Ref)**：**这是最强的功能！** 系统会自动读取当前战斗中所有敌人和环境的数据，生成纯文字版的简报（包含数值、武器、战术动机、特殊能力），战斗时无需翻书或切屏。
- **辅助工具**：
  - **倒计时 (Countdown)**：支持骰子/方格模式，可设置激活/推进/效果。
  - **计数器 (Counter)**：记录恐惧点、行动点等。
  - **自由布局**：所有小工具支持手动拖拽调整宽度。

### 🃏 卡片库 (Library)
- **批量导入**：支持拖拽导入由“敌人/场景卡生成器”导出的 JSON 文件（支持多选）。
- **多选管理**：支持批量删除、批量导出、批量修改卡片所属源（Core/Void/HB/自定义）。
- **强力筛选**：支持按 T级、来源、收藏状态 进行组合筛选。

## 📂 如何导入数据

本工具不包含预设的版权数据。你需要导入 JSON 文件来填充卡片库。
JSON 格式兼容**标准匕首之心卡片生成器**的导出格式。

1.  进入 **[卡片]** 标签页。
2.  点击右上角的 **📥 导入** 按钮。
3.  选择或拖入你的 `.json` 文件（支持一次性选择多个文件）。
4.  导入后，你可以使用 **✏️ 批量改所属** 功能来整理你的素材库。
